Nathalie's Personal Weight Lifting Circuit — Firebase Cloud Sync

To deploy:
1. Install Node.js.
2. Open Terminal.
3. Run: npm install -g firebase-tools
4. Run: firebase login
5. cd into this unzipped folder.
6. Run: firebase deploy --only hosting --project weight-lifting-circuit

Then open the Firebase Hosting URL on your phone and choose Add to Home Screen.
